Introduction

BoneMarrow.js is a lightweight, zero-dependency JavaScript library providing the structural core for reactive UIs. It gives you eight primitives — Model, Collection, Emitter, el(), View, Scope, createRefresh, and fetchJson — unified by a scope system that makes resource cleanup automatic and leak-free.

Unlike full frameworks, BoneMarrow has no virtual DOM, no compilation step, and no opinions about rendering. It runs in the browser and in Node.js / SSR environments.

Installation

BoneMarrow is available via npm and CDN. All exports live on the global bone object when loaded via script tag.

npm
npm install bonemarrow
CDN (jsDelivr)
cdn.jsdelivr.net/npm/bonemarrow

Script tag (browser)

index.html 
<script src="https://cdn.jsdelivr.net/npm/bonemarrow"></script>
<script>
  const scope = bone.createScope();
  const model = new bone.Model({ count: 0 }, scope);
</script>
import { createScope, Model } from "bonemarrow";

interface S { count: number; }
const scope = createScope();
const model = new Model<S>({ count: 0 }, scope);
ℹ️
The minified bundle is ~19kb with zero runtime dependencies. Ships both ESM and CJS builds.

Quick Start

Wire a Model, Collection, DOM bindings, and an event bus together — all cleaned up automatically when the page unloads.

app.js 
// 1. Create a scope
const scope = bone.createScope();

// 2. Model — reactive key/value state
const user = new bone.Model(
  { name: "Ada", score: 0 },
  scope
);

// 3. Collection — reactive list
const log = new bone.Collection([], scope);

// 4. Watch state changes
user.watch("score", score => {
  bone.el("#score").text(`Score: ${score}`);
}, scope);

// 5. Re-render list on any change
log.onChange(() => {
  bone.el("#log").html(
    log.map(e => `<li>${e}</li>`).join("")
  );
}, scope);

// 6. Bind a button
bone.el("#btn").on("click", () => {
  const next = user.get("score") + 1;
  user.set({ score: next });
  log.add(`score → ${next}`);
}, scope);
import { createScope, Model, Collection, el } from "bonemarrow";

interface UserState { name: string; score: number; }

const scope = createScope();

const user = new Model<UserState>(
  { name: "Ada", score: 0 }, scope
);

const log = new Collection<string>([], scope);

user.watch("score", (score) => {
  el("#score").text(`Score: ${score}`);
}, scope);

log.onChange(() => {
  el("#log").html(
    log.map((e: string) => `<li>${e}</li>`).join("")
  );
}, scope);

el("#btn").on("click", () => {
  const next = user.get("score") + 1;
  user.set({ score: next });
  log.add(`score → ${next}`);
}, scope);

Core Concepts

Scopes & Lifetimes

Every resource in BoneMarrow — event listeners, timers, watchers, fetch requests — is bound to a scope. When a scope is disposed, all its resources are cleaned up automatically. Scopes are hierarchical: disposing a parent disposes all children first.

AbortSignal integration

Each scope exposes a scope.signal — a native AbortSignal. Pass it directly to fetch(), addEventListener(), and any AbortSignal-aware API without any special wrappers.

signal.js 
const scope = bone.createScope();

// Native fetch — aborted when scope disposes
fetch("/api/data", { signal: scope.signal });

// Native addEventListener — same pattern
document.addEventListener("keydown", handler, {
  signal: scope.signal
});

scope.dispose(); // → fetch aborted, listener removed
import { createScope, Scope } from "bonemarrow";

const scope: Scope = createScope();

// scope.signal is typed as AbortSignal
fetch("/api/data", { signal: AbortSignal = scope.signal });

document.addEventListener("keydown", handler, {
  signal: scope.signal
});

scope.dispose();

Change detection

BoneMarrow uses strict equality (===) to detect changes. Calling set() or update() with the same values is a no-op — no events fire, no renders trigger.

💡
Best practice: Create one scope per component, page, or feature. Pass it down to models, collections, and event listeners. Dispose the scope when the feature is torn down.

🔭
Scope System

createScope(opts?) / createRootScope(opts?)

createScope(opts?: { debug?: boolean }) → Scope

createScope() creates a browser-environment scope attached to a shared global root that auto-disposes on pagehide and beforeunload. createRootScope() creates a standalone root for Node.js, Web Workers, or SSR.

Property Type Description
signal AbortSignal Native signal. Aborted when the scope disposes.
onDispose(fn) (fn: () => void) → void Register cleanup callback. Runs LIFO on disposal. Calls immediately if already disposed.
createChild() () → Scope Create a child scope. Disposed when parent disposes.
dispose() () → void Dispose this scope and all children. Safe to call multiple times.

isScopeDisposed(scope)

isScopeDisposed(scope: Scope) → boolean

Returns true if scope.signal.aborted is true.

scope.js 
// Browser scope (auto page-level cleanup)
const app = bone.createScope();

// Child scopes for sub-features
const modal  = app.createChild();
const widget = app.createChild();

// Register cleanup
modal.onDispose(() => {
  console.log("modal cleaned up");
});

// Pass signal to native APIs
document.addEventListener(
  "keydown", handler,
  { signal: modal.signal }
);

modal.dispose(); // app & widget still alive
app.dispose();   // widget disposed too

bone.isScopeDisposed(app); // → true

// Node.js / SSR / Worker
const root = bone.createRootScope({ debug: true });
import { createScope, createRootScope, isScopeDisposed, Scope } from "bonemarrow";

const app: Scope    = createScope();
const modal: Scope  = app.createChild();
const widget: Scope = app.createChild();

modal.onDispose(() => {
  console.log("modal cleaned up");
});

// scope.signal: AbortSignal — native API integration
document.addEventListener(
  "keydown", handler,
  { signal: modal.signal }
);

modal.dispose();
app.dispose();

isScopeDisposed(app); // → true

const root: Scope = createRootScope({ debug: true });

Model

A key-value state container with change tracking, per-key watchers, and optional JSON endpoint auto-refresh.

new Model(initialData, scope?) → Model

set / get / getAll

Method Returns Description
set(partial) boolean Merges partial into state. Emits change only for keys with new values. Returns true if any change occurred.
get(key) T[key] Returns current value for key.
getAll() T Returns a shallow copy of the entire state.

watch / onChange

Method Returns Description
watch(key, fn, scope?) () => void Calls fn(newValue) when the specific key changes. Returns an unsubscribe function.
onChange(fn, scope?) () => void Calls fn(changedPartial) on every set() that produces at least one change.

isDirty / has / reset

Method Returns Description
isDirty() boolean Returns true if any value differs from its initial value (strict equality).
has(key, value) boolean Returns true if data[key] === value.
reset() void Reverts all values to initial. Emits change if anything changed.

fetch / autoRefresh

autoRefresh.js 
const ticker = model.autoRefresh("/api/user", {
  interval:   10000,
  scope,
  immediate:  true,
  backoff:    true,
  maxRetries: 3,
  onError:    err => console.warn(err),
});

ticker.pause();
ticker.resume();
ticker.stop();
import { RefreshHandle } from "bonemarrow";

const ticker: RefreshHandle = model.autoRefresh("/api/user", {
  interval:   10000,
  scope,
  immediate:  true,
  backoff:    true,
  maxRetries: 3,
  onError:    (err: unknown) => console.warn(err),
});

ticker.pause();
ticker.resume();
ticker.stop();
model.js 
const scope = bone.createScope();

// Create with initial state
const user = new bone.Model({
  name:   "Ada",
  score:  0,
  active: true,
}, scope);

// Watch a specific key
user.watch("score", score => {
  bone.el("#score").text(`Score: ${score}`);
}, scope);

// Listen to all changes
user.onChange(changed => {
  console.log("changed:", changed);
}, scope);

// Update — only changed keys emit
user.set({ score: 42 }); // → fires "score" watcher
user.set({ score: 42 }); // → no-op, same value

// Dirty tracking
user.isDirty(); // → true (score changed)
user.reset();   // → score back to 0

// Auto-poll an API
user.autoRefresh("/api/user", {
  interval:   10000,
  scope,
  backoff:    true,
  maxRetries: 3,
});
import { createScope, Model, el } from "bonemarrow";

interface UserState {
  name:   string;
  score:  number;
  active: boolean;
}

const scope = createScope();
const user = new Model<UserState>({
  name: "Ada", score: 0, active: true,
}, scope);

// score is inferred as number
user.watch("score", (score) => {
  el("#score").text(`Score: ${score}`);
}, scope);

user.onChange((changed: Partial<UserState>) => {
  console.log("changed:", changed);
}, scope);

user.set({ score: 42 });
user.isDirty(); // → true
user.reset();

user.autoRefresh("/api/user", {
  interval: 10000, scope, backoff: true, maxRetries: 3,
});

Collection

A reactive array that emits granular events on mutation. All standard array read methods are non-mutating.

new Collection(initialItems?, scope?) → Collection

add / remove / removeAt / update

Method Returns Description
add(...items) void Pushes items to end. Emits add.
remove(predicate) T[] Removes matching items. Emits remove.
removeAt(index) T | undefined Removes item at index. Emits remove.
update(predicate, patch) T[] Shallow-merges patch into matching items. Only emits update if values changed.

sort / move / reset / clear

Method Returns Description
sort(compareFn) void Sorts in-place. Emits sort.
move(from, to) void Moves item from index to index. Emits sort.
reset(items) void Replaces all items. Emits reset.
clear() void Empties the collection. Emits reset with [].

Event subscriptions

Method Payload Description
onAdd(fn, scope?) T[] Fires with added items.
onRemove(fn, scope?) T[] Fires with removed items.
onUpdate(fn, scope?) T[] Fires with updated items.
onReset(fn, scope?) T[] Fires with new full array.
onChange(fn, scope?) void Fires on any mutation.
collection.js 
const scope = bone.createScope();

const tasks = new bone.Collection([
  { id: 1, title: "Design", done: false },
  { id: 2, title: "Build",  done: false },
], scope);

// Re-render whenever anything changes
tasks.onChange(() => renderList(), scope);

// Add items
tasks.add({ id: 3, title: "Ship", done: false });

// Mark a task complete
tasks.update(
  t => t.id === 1,
  { done: true }
);

// Remove done tasks
tasks.remove(t => t.done);

// Sort alphabetically
tasks.sort((a, b) =>
  a.title.localeCompare(b.title)
);

// Granular event subscriptions
tasks.onAdd(items => {
  console.log("added:", items);
}, scope);

// Read without mutating
const pending = tasks.filter(t => !t.done);
console.log(tasks.length); // → 2
import { createScope, Collection } from "bonemarrow";

interface Task { id: number; title: string; done: boolean; }

const scope = createScope();
const tasks = new Collection<Task>([
  { id: 1, title: "Design", done: false },
  { id: 2, title: "Build",  done: false },
], scope);

tasks.onChange(() => renderList(), scope);

tasks.add({ id: 3, title: "Ship", done: false });

tasks.update(
  (t: Task) => t.id === 1,
  { done: true }
);

tasks.remove((t: Task) => t.done);

tasks.sort((a: Task, b: Task) =>
  a.title.localeCompare(b.title)
);

tasks.onAdd((items: Task[]) => {
  console.log("added:", items);
}, scope);

const pending: Task[] = tasks.filter(t => !t.done);
console.log(tasks.length);

Emitter / TypedEmitter

A full-featured event bus. TypedEmitter<Events> adds compile-time type-safety for event names and payloads.

new Emitter(opts?) → Emitter
Option Type Default Description
debug boolean false Logs listener add/remove and emissions to console.
maxListeners number 50 Warns when any event exceeds this many listeners.

on / once / off

Method Returns Description
on(event, fn, scope?) () => void Subscribe. Returns unsubscribe. Scope auto-removes on disposal.
once(event, fn, scope?) () => void Subscribe once — auto-removes after first emission.
off(event) void Removes all listeners for the event.

emit / emitAsync

Method Returns Description
emit(event, ...args) void Calls all handlers synchronously. Errors caught individually.
emitAsync(event, ...args) Promise<void> Awaits all handlers via Promise.allSettled. Throws AggregateError on failure.

onceAsync

onceAsync(event, scope?) → Promise<args[]>

Returns a Promise resolving with the event arguments array on the next emission.

emitter.js 
const bus = new bone.Emitter({
  debug:        true,
  maxListeners: 20,
});

// Standard subscribe
bus.on("login", ({ user }) => {
  console.log("logged in:", user);
}, scope);

// One-time listener
bus.once("ready", () => init());

// Await next emission
async function waitForData() {
  const [payload] = await bus.onceAsync("data");
  console.log(payload);
}

// Synchronous emit
bus.emit("login", { user: "Ada" });

// Async emit — waits all handlers
await bus.emitAsync("save", { id: 42 });

bus.listenerCount("login"); // → 1
bus.eventNames();          // → ["login", "ready"]
import { TypedEmitter } from "bonemarrow";

type Events = {
  login: [{ user: string }];
  ready: [];
  data:  [unknown];
  save:  [{ id: number }];
};

const bus = new TypedEmitter<Events>({
  debug: true, maxListeners: 20,
});

// user is string — type-checked ✓
bus.on("login", ({ user }) => {
  console.log("logged in:", user);
}, scope);

bus.once("ready", () => init());

async function waitForData() {
  const [payload] = await bus.onceAsync("data");
  console.log(payload);
}

bus.emit("login", { user: "Ada" });
await bus.emitAsync("save", { id: 42 });
bus.listenerCount("login");
bus.eventNames();

🎯
el() & Elements

el() wraps one or more DOM elements in a chainable Elements instance. All methods apply to every matched element. Getters operate on the first matched element.

el(input: string | Element | NodeList | Element[]) → Elements

Content, classes & attributes

Method Description
text(str?) / html(str?) Get or set textContent / innerHTML.
val(str?) Get or set value for input, textarea, select.
addClass / removeClass / toggleClass / hasClass Space-separated class names. toggleClass(name, force?) accepts optional boolean.
css(prop, val?) / css(object) Get or set CSS custom properties. Accepts object map.
attr(key, val?) / removeAttr(key) Get or set HTML attributes. data(key, val?) works with dataset.

Events

Method Description
on(event, fn, scope?) Add listener to all matched elements. Scope auto-removes on disposal.
once(event, fn) Add a one-time listener.
trigger(event, detail?, opts?) Dispatch a CustomEvent. opts accepts bubbles and cancelable.

Traversal & mutation

Traversal: find, filter, not, parent, siblings, children, closest — all return new Elements instances. Mutation: append, prepend, remove, empty, clone, show, hide, toggle, focus.

el.js 
// Select & chain
bone.el("#btn")
  .addClass("active")
  .text("Clicked!")
  .css("--color", "red");

// Scope-bound event listener
bone.el("#btn").on("click", e => {
  // auto-removed on scope.dispose()
}, scope);

// Traverse the DOM
bone.el(".list")
  .find("li")
  .filter(".active")
  .each(el => {
    bone.el(el).addClass("highlight");
  });

// Create from HTML string
const card = bone.elFromHtml(
  `<div class="card">Hello</div>`
);
bone.el("#container").append(card);

// Dispatch custom events
bone.el("#form").trigger("submit", {
  data: { name: "Ada" }
});
import { el, elFromHtml, Elements } from "bonemarrow";

// el() returns a typed Elements instance
const btn: Elements = el("#btn");

btn
  .addClass("active")
  .text("Clicked!")
  .css("--color", "red");

btn.on("click", (e: Event) => {}, scope);

el(".list")
  .find("li")
  .filter(".active")
  .each((node: Element) => {
    el(node).addClass("highlight");
  });

const card: Elements = elFromHtml(
  `<div class="card">Hello</div>`
);
el("#container").append(card);

el("#form").trigger("submit", {
  data: { name: "Ada" }
});

View

A base class for component-style DOM views. Extend it to encapsulate a root element, its model, and its child views — all scope-managed.

class MyView extends bone.View

Lifecycle & query

Method / Property Description
constructor(root, model, opts?) root: DOM Element. autoDestroy (default true): MutationObserver-based auto-dispose when root removed.
init() Override to set up listeners and render. Called once in constructor after scope is ready.
render() Optional render hook. Call manually or from watchers.
$(selector) Query within root. Returns Elements.
$root() Root element wrapped in Elements.
on / onDocument / onWindow Scope-bound event listeners — auto-removed on dispose.
emit(event, detail?) Dispatch a bubbling CustomEvent from root.

Child views

Method Description
createChild(ViewClass, el, model?) Instantiate a child view with scope derived from parent. Auto-destroyed when parent is destroyed.
createChildren(ViewClass, selector, modelFn?) Create a child view for each element matching selector. modelFn(el, index) provides per-element models.
view.js 
class CounterView extends bone.View {

  init() {
    this.model.watch("count", v => {
      this.$(".count").text(v);
    }, this.scope);

    this.on("click", () => {
      this.model.set({
        count: this.model.get("count") + 1
      });
    });
  }

  render() {
    const { count } = this.model.getAll();
    this.$root().html(
      `<span class="count">${count}</span>`
    );
  }
}

// Mount with createView helper
const model = new bone.Model({ count: 0 });
bone.createView(CounterView, "#counter", model);

// Create child views
class AppView extends bone.View {
  init() {
    this.createChildren(CounterView, ".counter");
  }
}
import { View, Model, createView } from "bonemarrow";

interface CounterState { count: number; }

class CounterView extends View<Model<CounterState>> {

  init(): void {
    this.model.watch("count", (v) => {
      this.$(".count").text(String(v));
    }, this.scope);

    this.on("click", () => {
      this.model.set({
        count: this.model.get("count") + 1
      });
    });
  }

  render(): void {
    const { count } = this.model.getAll();
    this.$root().html(
      `<span class="count">${count}</span>`
    );
  }
}

const model = new Model<CounterState>({ count: 0 });
createView(CounterView, "#counter", model);

fetchJson

Scope-aware JSON fetcher with timeout, retries, exponential backoff, and in-flight request deduplication.

fetchJson(url, opts?) → Promise<T>
Option Type Default Description
scope Scope — Aborts the request if scope disposes.
timeout number — Milliseconds before aborting. Applied per attempt.
retryOnFailure number 0 Additional attempts on non-abort errors.
retryDelay number 0 Milliseconds between retry attempts.
dedupe boolean false Same Promise for identical concurrent calls within the same scope.
init RequestInit — Passed to native fetch().
parse (data: any) => T — Optional transform applied to parsed JSON before resolving.
fetchJson.js 
// Simple fetch with scope
const user = await bone.fetchJson(
  "/api/user/1",
  { scope }
);

// With timeout & retries
const data = await bone.fetchJson(
  "/api/data",
  {
    scope,
    timeout:        5000,
    retryOnFailure: 3,
    retryDelay:     1000,
  }
);

// Deduplicated — same Promise returned
bone.fetchJson("/api/config", { scope, dedupe: true });
bone.fetchJson("/api/config", { scope, dedupe: true });
// → only ONE network request made

// POST with custom init
const result = await bone.fetchJson(
  "/api/save",
  {
    scope,
    init: {
      method:  "POST",
      headers: { "Content-Type": "application/json" },
      body:    JSON.stringify({ name: "Ada" }),
    },
    parse: res => res.data,
  }
);
import { fetchJson } from "bonemarrow";

interface User { id: number; name: string; }

const user = await fetchJson<User>(
  "/api/user/1",
  { scope }
);
// user.name is string ✓

const data = await fetchJson<User[]>(
  "/api/data",
  {
    scope,
    timeout:        5000,
    retryOnFailure: 3,
    retryDelay:     1000,
  }
);

// Deduplicated within same scope
fetchJson("/api/config", { scope, dedupe: true });
fetchJson("/api/config", { scope, dedupe: true });

const result = await fetchJson<User>(
  "/api/save",
  {
    scope,
    init: { method: "POST", body: JSON.stringify({ name: "Ada" }) },
    parse: (res: { data: User }) => res.data,
  }
);

⏱
createRefresh

An interval-based async poller. Waits for each tick to complete before scheduling the next, skips overlapping ticks, and supports pause/resume and backoff.

createRefresh(fn, opts) → RefreshHandle
Option Type Default Description
interval * number — Milliseconds between end of one tick and start of next.
scope * Scope — Poller stops when scope disposes.
immediate boolean true Run first tick immediately.
startPaused boolean false Create paused — call resume() to start.
backoff boolean false Double interval after each error (capped 10×). Resets on success.
maxRetries number 0 Stop after this many consecutive errors. 0 = unlimited.
💡
Skip overlap: If isExecuting() is true when the next interval fires, the tick is silently skipped.
refresh.js 
const poller = bone.createRefresh(
  async (signal) => {
    const data = await fetch("/api/feed", {
      signal
    }).then(r => r.json());
    feed.reset(data);
  },
  {
    interval:   5000,
    scope,
    immediate:  true,
    backoff:    true,
    maxRetries: 5,
    onError:    err => console.warn(err),
  }
);

// Pause when tab is hidden
document.addEventListener("visibilitychange", () => {
  document.hidden
    ? poller.pause()
    : poller.resume();
});

poller.isPaused();    // → false
poller.isExecuting(); // → true/false
poller.stop();
import { createRefresh, RefreshHandle, Collection } from "bonemarrow";

const poller: RefreshHandle = createRefresh(
  async (signal: AbortSignal) => {
    const data = await fetch("/api/feed", { signal })
      .then((r) => r.json());
    feed.reset(data);
  },
  {
    interval:   5000,
    scope,
    immediate:  true,
    backoff:    true,
    maxRetries: 5,
    onError:    (err: unknown) => console.warn(err),
  }
);

document.addEventListener("visibilitychange", () => {
  document.hidden ? poller.pause() : poller.resume();
});

poller.isPaused();
poller.isExecuting();
poller.stop();
