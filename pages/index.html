BoneMarrow.js gives you Models, Collections, Views, Emitters, and scope-aware utilities — everything to build reactive UIs without a heavyweight framework.

Read the Docs →
app.js 
// Scope-bound reactive model
const scope = bone.createScope();
const user  = new bone.Model({ name: "Ada", score: 0 }, scope);

// Watch a specific key
user.watch("score", val => {
  bone.el("#score").text(`Score: ${val}`);
}, scope);

// Trigger change → watcher fires
user.set({ score: 42 });
import { createScope, Model, el } from "bonemarrow";

interface UserState { name: string; score: number; }

const scope = createScope();
const user = new Model<UserState>({ name: "Ada", score: 0 }, scope);

// val is inferred as number
user.watch("score", (val) => {
  el("#score").text(`Score: ${val}`);
}, scope);

user.set({ score: 42 });

What's inside

Every piece. No bloat.

BoneMarrow gives you the structural primitives for state, events, and DOM — all scope-aware and leak-free by design.

📦
Model
Key-value state with dirty-tracking, per-key watch(), and autoRefresh() from any JSON endpoint.
🗂
Collection
Array-backed list with add/remove/update/sort/reset events. All standard array methods plus reactive hooks.
📡
Emitter & TypedEmitter
Full-featured typed event bus with on, once, onceAsync, emit, and emitAsync. Listener leak detection built in.
🎯
el() & Elements
Fluent DOM API. Select, traverse, mutate, and bind events with a chainable wrapper — no jQuery required.
🧱
View
Component base class with child management, scope inheritance, and MutationObserver-powered auto-destroy.
⏱
createRefresh
Interval-based poller with pause/resume, exponential backoff, max retries, and clean abort on scope disposal.
🔭
Scope System
Hierarchical lifetimes via createScope / createRootScope. Disposal cascades to children and auto-cleans listeners and timers.
🌐
fetchJson
Scope-aware fetch with timeout, retries, exponential backoff, and request deduplication. Cancels on scope disposal.
