Model
Key-value state container with change tracking, per-key watchers, and endpoint auto-refresh.
✓ ✓ Scope-aware✓ ✓ Dirty tracking✓ ✓ Per-key watch✓ ✓ Auto-refreshTypeScript generic
new Model(data, scope?)
Constructor
Creates a model with initial data. Pass a scope for lifetime binding.
set(partial)
Set values
Merges partial data into state. Only emits change for keys whose values actually changed. Returns true if any change occurred.
get(key)
Read a value
Returns the current value for a single key. Type-safe with TypeScript generics.
getAll()
Read all values
Returns a shallow copy of the full state object.
watch(key, fn, scope?)
Per-key watcher
Calls fn(newValue) whenever the key changes. Auto-removed on scope disposal.
onChange(fn, scope?)
Global change listener
Fires with the partial change object { key: newValue } for every set() that produces a change.
isDirty()
Dirty check
Returns true if any current value differs from its initial value.
reset()
Reset to initial
Reverts all values to the initial state. Emits change if anything changed.
fetch(url, timeout?)
Load from endpoint
Fetches a JSON endpoint and calls set() with the result.
autoRefresh(url, opts)
Polling refresh
Polls a JSON endpoint on an interval. Supports backoff, retries, and pause/resume.
Note: Model emits change events only for keys that actually changed (strict equality). Calling set() with the same data is a safe no-op.
model.js 
const scope = bone.createScope();

// Create with initial state
const user = new bone.Model({
  name:   "Ada",
  score:  0,
  active: true,
}, scope);

// Watch a specific key
user.watch("score", score => {
  bone.el("#score").text(`Score: ${score}`);
}, scope);

// Listen to all changes
user.onChange(changed => {
  console.log("changed:", changed);
}, scope);

// Update — only changed keys emit
user.set({ score: 42 }); // → fires "score" watcher
user.set({ score: 42 }); // → no-op, same value

// Dirty tracking
user.isDirty(); // → true (score changed)
user.reset();   // → score back to 0

// Auto-poll an API
user.autoRefresh("/api/user", {
  interval:   10000,
  scope,
  backoff:    true,
  maxRetries: 3,
});
import { createScope, Model, el } from "bonemarrow";

interface UserState {
  name:   string;
  score:  number;
  active: boolean;
}

const scope = createScope();
const user = new Model<UserState>({
  name: "Ada", score: 0, active: true,
}, scope);

// score is inferred as number
user.watch("score", (score) => {
  el("#score").text(`Score: ${score}`);
}, scope);

user.onChange((changed: Partial<UserState>) => {
  console.log("changed:", changed);
}, scope);

user.set({ score: 42 });
user.isDirty(); // → true
user.reset();

user.autoRefresh("/api/user", {
  interval: 10000, scope, backoff: true, maxRetries: 3,
});

Collection
Reactive array with granular add / remove / update / sort events and full array-method API.
✓ ✓ Scope-aware✓ ✓ Granular events✓ ✓ Auto-refreshWorks with any type
add(...items)
Add items
Pushes one or more items to the end of the collection. Emits add with the array of added items.
remove(predicate)
Remove by predicate
Removes all items matching the predicate. Emits remove with removed items.
removeAt(index)
Remove by index
Removes the item at the given index. Returns the removed item.
update(predicate, patch)
Patch matching items
Shallow-merges patch onto matching items. Only emits update if values changed.
sort(compareFn)
Sort in-place
Sorts the internal array. Emits sort.
move(fromIndex, toIndex)
Reorder item
Moves an item from one index to another. Emits sort. Useful for drag-and-drop.
reset(items)
Replace all
Replaces the entire collection with a new array. Emits reset.
clear()
Empty collection
Removes all items and emits reset with [].
onAdd / onRemove / onUpdate / onReset / onSort
Granular event hooks
Subscribe to individual mutation events. Each returns an unsubscribe function.
map / filter / find / some / every / forEach
Array methods
Standard array iteration methods operating on a snapshot of the internal array. Non-mutating.
Note: update() performs shallow equality checks before emitting — patching an item with identical values is a no-op.
collection.js 
const scope = bone.createScope();

const tasks = new bone.Collection([
  { id: 1, title: "Design", done: false },
  { id: 2, title: "Build",  done: false },
], scope);

// Re-render whenever anything changes
tasks.onChange(() => renderList(), scope);

// Add items
tasks.add({ id: 3, title: "Ship", done: false });

// Mark a task complete
tasks.update(
  t => t.id === 1,
  { done: true }
);

// Remove done tasks
tasks.remove(t => t.done);

// Sort alphabetically
tasks.sort((a, b) =>
  a.title.localeCompare(b.title)
);

// Granular event subscriptions
tasks.onAdd(items => {
  console.log("added:", items);
}, scope);

// Read without mutating
const pending = tasks.filter(t => !t.done);
console.log(tasks.length); // → 2
import { createScope, Collection } from "bonemarrow";

interface Task { id: number; title: string; done: boolean; }

const scope = createScope();
const tasks = new Collection<Task>([
  { id: 1, title: "Design", done: false },
  { id: 2, title: "Build",  done: false },
], scope);

tasks.onChange(() => renderList(), scope);

tasks.add({ id: 3, title: "Ship", done: false });

tasks.update(
  (t: Task) => t.id === 1,
  { done: true }
);

tasks.remove((t: Task) => t.done);

tasks.sort((a: Task, b: Task) =>
  a.title.localeCompare(b.title)
);

tasks.onAdd((items: Task[]) => {
  console.log("added:", items);
}, scope);

const pending: Task[] = tasks.filter(t => !t.done);
console.log(tasks.length);

Emitter / TypedEmitter
Full-featured event bus with async emission, promise-based once, and built-in leak detection.
✓ ✓ Typed events✓ ✓ Async emit✓ ✓ Scope cleanup✓ ✓ Leak detection
on(event, fn, scope?)
Subscribe
Registers a listener. Returns an unsubscribe function. Pass a scope to auto-remove on disposal.
once(event, fn, scope?)
One-time listener
Like on() but auto-removes after the first emission.
onceAsync(event, scope?)
Promise-based once
Returns a Promise resolving with the event arguments array on the next emission.
emit(event, ...args)
Synchronous emit
Calls all listeners synchronously. Errors in handlers are caught and logged without stopping other handlers.
emitAsync(event, ...args)
Async emit
Calls all handlers and awaits via Promise.allSettled. Throws AggregateError if any handler rejects.
off(event)
Remove all for event
Removes all listeners for the given event name.
listenerCount(event)
Count listeners
Returns the number of active listeners for the event.
eventNames()
List events
Returns an array of all event names with active listeners.
Note: Use TypedEmitter<Events> for full TypeScript type-safety on event names and payload types. Emitter is the untyped alias for plain JS.
emitter.js 
const bus = new bone.Emitter({
  debug:        true,
  maxListeners: 20,
});

// Standard subscribe
bus.on("login", ({ user }) => {
  console.log("logged in:", user);
}, scope);

// One-time listener
bus.once("ready", () => init());

// Await next emission
async function waitForData() {
  const [payload] = await bus.onceAsync("data");
  console.log(payload);
}

// Synchronous emit
bus.emit("login", { user: "Ada" });

// Async emit — waits all handlers
await bus.emitAsync("save", { id: 42 });

bus.listenerCount("login"); // → 1
bus.eventNames();          // → ["login", "ready"]
import { TypedEmitter } from "bonemarrow";

type Events = {
  login: [{ user: string }];
  ready: [];
  data:  [unknown];
  save:  [{ id: number }];
};

const bus = new TypedEmitter<Events>({
  debug: true, maxListeners: 20,
});

// user is string — type-checked ✓
bus.on("login", ({ user }) => {
  console.log("logged in:", user);
}, scope);

bus.once("ready", () => init());

async function waitForData() {
  const [payload] = await bus.onceAsync("data");
  console.log(payload);
}

bus.emit("login", { user: "Ada" });
await bus.emitAsync("save", { id: 42 });
bus.listenerCount("login");
bus.eventNames();

el() & Elements
Chainable DOM wrapper. Select, traverse, mutate, animate, and bind events — no dependencies.
✓ ✓ Chainable✓ ✓ Multi-element✓ ✓ Scope eventsjQuery-inspired
el(selector | Element | NodeList)
Wrap elements
Accepts a CSS selector, DOM Element, NodeList, or array. Returns a chainable Elements instance.
text(str?) / html(str?)
Content
Get or set textContent / innerHTML. Getter from first element; setter applies to all.
addClass / removeClass / toggleClass / hasClass
Class manipulation
Space-separated class names supported. toggleClass(name, force?) accepts an optional boolean.
css(prop, val) / css(object)
Inline styles
Set a CSS property or multiple via an object map. Getter returns computed style from first element.
attr(key, val?) / removeAttr(key)
Attributes
Get or set HTML attributes. data(key, val?) works with dataset.
on(event, fn, scope?) / off / once / trigger
Events
Bind scope-aware listeners. trigger() dispatches a CustomEvent with optional detail payload.
find / filter / not / parent / siblings / closest / children
DOM traversal
All traversal methods return a new Elements instance. Chain freely.
append / prepend / remove / empty / clone
DOM mutation
Insert HTML strings or Elements. Nodes auto-cloned when appending to multiple targets.
show / hide / toggle / isVisible
Visibility
Controls style.display. toggle(force?) accepts an optional boolean.
Note: elFromHtml(str) creates an Elements instance directly from an HTML string via a <template> element — no DOM insertion needed.
el.js 
// Select & chain
bone.el("#btn")
  .addClass("active")
  .text("Clicked!")
  .css("--color", "red");

// Scope-bound event listener
bone.el("#btn").on("click", e => {
  // auto-removed on scope.dispose()
}, scope);

// Traverse the DOM
bone.el(".list")
  .find("li")
  .filter(".active")
  .each(el => {
    bone.el(el).addClass("highlight");
  });

// Create from HTML string
const card = bone.elFromHtml(
  `<div class="card">Hello</div>`
);
bone.el("#container").append(card);

// Dispatch custom events
bone.el("#form").trigger("submit", {
  data: { name: "Ada" }
});
import { el, elFromHtml, Elements } from "bonemarrow";

// el() returns a typed Elements instance
const btn: Elements = el("#btn");

btn
  .addClass("active")
  .text("Clicked!")
  .css("--color", "red");

btn.on("click", (e: Event) => {}, scope);

el(".list")
  .find("li")
  .filter(".active")
  .each((node: Element) => {
    el(node).addClass("highlight");
  });

const card: Elements = elFromHtml(
  `<div class="card">Hello</div>`
);
el("#container").append(card);

el("#form").trigger("submit", {
  data: { name: "Ada" }
});

View
Component base class with root element binding, child management, and MutationObserver auto-destroy.
✓ ✓ Auto-destroy✓ ✓ Child views✓ ✓ Scope-inheritedExtend to use
constructor(root, model, opts?)
Constructor
Binds to a root DOM element and optional model. Set autoDestroy: false to disable MutationObserver teardown.
init()
Lifecycle hook
Called once in constructor after scope setup. Override to wire event listeners and initial render logic.
render()
Render hook
Optional override for declarative render logic. Call manually or from model change listeners.
$(selector)
Scoped query
Queries within the view's root element. Returns an Elements instance.
$root()
Root element
Returns the view's root wrapped in Elements.
on(event, fn) / onDocument / onWindow
Scoped listeners
Listeners automatically removed when the view's scope disposes.
createChild(ViewClass, el, model?)
Create child view
Child scope is derived from parent. Child is destroyed when parent is destroyed.
createChildren(ViewClass, selector, modelFn?)
Create multiple children
Creates a child view for each element matching the selector. modelFn(el, i) provides per-element models.
emit(event, detail?)
Dispatch event
Dispatches a bubbling CustomEvent from the root element.
Note: When autoDestroy is enabled (default), a MutationObserver watches the root's parent. If the root is removed from the DOM, the view's scope disposes automatically.
view.js 
class CounterView extends bone.View {

  init() {
    this.model.watch("count", v => {
      this.$(".count").text(v);
    }, this.scope);

    this.on("click", () => {
      this.model.set({
        count: this.model.get("count") + 1
      });
    });
  }

  render() {
    const { count } = this.model.getAll();
    this.$root().html(
      `<span class="count">${count}</span>`
    );
  }
}

// Mount with createView helper
const model = new bone.Model({ count: 0 });
bone.createView(CounterView, "#counter", model);

// Create child views
class AppView extends bone.View {
  init() {
    this.createChildren(CounterView, ".counter");
  }
}
import { View, Model, createView } from "bonemarrow";

interface CounterState { count: number; }

class CounterView extends View<Model<CounterState>> {

  init(): void {
    this.model.watch("count", (v) => {
      this.$(".count").text(String(v));
    }, this.scope);

    this.on("click", () => {
      this.model.set({
        count: this.model.get("count") + 1
      });
    });
  }

  render(): void {
    const { count } = this.model.getAll();
    this.$root().html(
      `<span class="count">${count}</span>`
    );
  }
}

const model = new Model<CounterState>({ count: 0 });
createView(CounterView, "#counter", model);

Scope System
Hierarchical lifetime management. Disposal cascades to all children, cleaning listeners, timers, and resources.
✓ ✓ Hierarchical✓ ✓ AbortSignal-backed✓ ✓ Browser & Node.js✓ ✓ Leak detection
createScope(opts?)
Browser scope
Creates a scope attached to a shared browser root. The root auto-disposes on pagehide and beforeunload.
createRootScope(opts?)
Root scope
Creates a standalone root scope. Use in Node.js, Web Workers, or SSR where window is not available.
scope.createChild()
Child scope
Creates a child scope. Disposing the parent disposes all children first.
scope.onDispose(fn)
Cleanup callback
Registers a cleanup function to run on disposal. Runs in reverse registration order (LIFO).
scope.dispose()
Dispose
Disposes this scope and all children. Aborts scope.signal. Safe to call multiple times.
scope.signal
AbortSignal
The underlying AbortSignal. Pass directly to fetch(), addEventListener(), and any AbortSignal-aware API.
isScopeDisposed(scope)
Check disposed
Returns true if scope.signal.aborted is true.
Note: Pass { debug: true } to get console warnings when more than 50 cleanup callbacks are registered on a single scope — a signal of potential leaks.
scope.js 
// Browser scope (auto page-level cleanup)
const app = bone.createScope();

// Child scopes for sub-features
const modal  = app.createChild();
const widget = app.createChild();

// Register cleanup
modal.onDispose(() => {
  console.log("modal cleaned up");
});

// Pass signal to native APIs
document.addEventListener(
  "keydown", handler,
  { signal: modal.signal }
);

modal.dispose(); // app & widget still alive
app.dispose();   // widget disposed too

bone.isScopeDisposed(app); // → true

// Node.js / SSR / Worker
const root = bone.createRootScope({ debug: true });
import { createScope, createRootScope, isScopeDisposed, Scope } from "bonemarrow";

const app: Scope    = createScope();
const modal: Scope  = app.createChild();
const widget: Scope = app.createChild();

modal.onDispose(() => {
  console.log("modal cleaned up");
});

// scope.signal: AbortSignal — native API integration
document.addEventListener(
  "keydown", handler,
  { signal: modal.signal }
);

modal.dispose();
app.dispose();

isScopeDisposed(app); // → true

const root: Scope = createRootScope({ debug: true });

createRefresh
Interval-based async poller with pause/resume, exponential backoff, and scope-aware cleanup.
✓ ✓ Pause / Resume✓ ✓ Exponential backoff✓ ✓ Max retries✓ ✓ Skip overlap
createRefresh(fn, opts)
Create a poller
Runs fn(signal) on an interval. The signal is aborted if the poller stops mid-execution.
opts.interval
Interval (ms)
Required. Time between the end of one tick and the start of the next.
opts.immediate
Run immediately
Default true. If true, the first tick fires immediately rather than waiting for the first interval.
opts.startPaused
Start paused
If true, the poller is created but doesn't start until resume() is called.
opts.backoff
Exponential backoff
Doubles the interval after each consecutive error (capped at 10×). Resets to base on success.
opts.maxRetries
Max consecutive errors
Stops the poller automatically after this many consecutive errors.
pause() / resume() / stop()
Control
pause() halts ticking; resume() restarts it. stop() permanently stops and aborts any in-flight tick.
isPaused() / isExecuting()
State inspection
Returns current poller state. A tick is skipped if a previous one is still running.
Note: If a tick is still running when the next interval fires, the new tick is silently skipped — no piling up of concurrent requests.
refresh.js 
const poller = bone.createRefresh(
  async (signal) => {
    const data = await fetch("/api/feed", {
      signal
    }).then(r => r.json());
    feed.reset(data);
  },
  {
    interval:   5000,
    scope,
    immediate:  true,
    backoff:    true,
    maxRetries: 5,
    onError:    err => console.warn(err),
  }
);

// Pause when tab is hidden
document.addEventListener("visibilitychange", () => {
  document.hidden
    ? poller.pause()
    : poller.resume();
});

poller.isPaused();    // → false
poller.isExecuting(); // → true/false
poller.stop();
import { createRefresh, RefreshHandle, Collection } from "bonemarrow";

const poller: RefreshHandle = createRefresh(
  async (signal: AbortSignal) => {
    const data = await fetch("/api/feed", { signal })
      .then((r) => r.json());
    feed.reset(data);
  },
  {
    interval:   5000,
    scope,
    immediate:  true,
    backoff:    true,
    maxRetries: 5,
    onError:    (err: unknown) => console.warn(err),
  }
);

document.addEventListener("visibilitychange", () => {
  document.hidden ? poller.pause() : poller.resume();
});

poller.isPaused();
poller.isExecuting();
poller.stop();

fetchJson
Scope-aware JSON fetcher with timeout, retries, backoff, and request deduplication.
✓ ✓ Auto-cancel on dispose✓ ✓ Dedupe✓ ✓ Retry + backoff✓ ✓ Timeout
fetchJson(url, opts?)
Fetch JSON
Fetches the URL, parses as JSON, and returns it. Aborted when the scope disposes.
opts.scope
Scope binding
Aborts the request and all retries if the scope disposes mid-flight.
opts.timeout
Request timeout
Milliseconds before aborting. Applied per attempt when combined with retries.
opts.retryOnFailure
Retry count
Number of additional attempts on failure. Does not retry on AbortError.
opts.retryDelay
Delay between retries
Milliseconds to wait between retry attempts. The delay is also abortable.
opts.dedupe
Request deduplication
When true, identical in-flight requests within the same scope share the same Promise.
opts.parse
Custom parser
Optional transform function applied to the parsed JSON before resolving.
opts.init
Fetch init
Passed directly to native fetch(). Set method, headers, body, credentials, etc.
Note: The dedupe cache is keyed per-scope instance. Two different scopes fetching the same URL will not share a request.
fetchJson.js 
// Simple fetch with scope
const user = await bone.fetchJson(
  "/api/user/1",
  { scope }
);

// With timeout & retries
const data = await bone.fetchJson(
  "/api/data",
  {
    scope,
    timeout:        5000,
    retryOnFailure: 3,
    retryDelay:     1000,
  }
);

// Deduplicated — same Promise returned
bone.fetchJson("/api/config", { scope, dedupe: true });
bone.fetchJson("/api/config", { scope, dedupe: true });
// → only ONE network request made

// POST with custom init
const result = await bone.fetchJson(
  "/api/save",
  {
    scope,
    init: {
      method:  "POST",
      headers: { "Content-Type": "application/json" },
      body:    JSON.stringify({ name: "Ada" }),
    },
    parse: res => res.data,
  }
);
import { fetchJson } from "bonemarrow";

interface User { id: number; name: string; }

const user = await fetchJson<User>(
  "/api/user/1",
  { scope }
);
// user.name is string ✓

const data = await fetchJson<User[]>(
  "/api/data",
  {
    scope,
    timeout:        5000,
    retryOnFailure: 3,
    retryDelay:     1000,
  }
);

// Deduplicated within same scope
fetchJson("/api/config", { scope, dedupe: true });
fetchJson("/api/config", { scope, dedupe: true });

const result = await fetchJson<User>(
  "/api/save",
  {
    scope,
    init: { method: "POST", body: JSON.stringify({ name: "Ada" }) },
    parse: (res: { data: User }) => res.data,
  }
);
